# OnlineJudge

顧名思義,是一個線上評測系統,  
給誰使用的呢?
這是專為程式設計所做的驗證評測系統,利用各式不同的題型,
驗證程式輸出輸入藉以判別學習者所建立的程式碼是否正確,
並根據設定給予分數  
今天就以  
青島大學OnlineJudge線上評測系統  
作為範例做一個介紹，青島大學的OnlineJudge，主要有幾個特色:  
* Docker一鍵部署
* 前後端分離
* ACM/OI兩種模式
* 多語言支持(C , Python, Java)
